# Contributing

This is a personal project by [@tsuimou](https://github.com/tsuimou).

## Want to contribute?

I'm open to contributions! Here's how:

1. **Open an issue first** - Describe what you want to add or fix
2. **Fork the repo** - Make your changes in a branch
3. **Submit a PR** - Link it to the issue

## Guidelines

- Keep it simple and human-first
- No technical jargon in the UI
- Test your changes with `python3 -m claude_monitor`

## Not accepting

- Major architectural changes without discussion
- Features that add complexity without clear benefit

## Questions?

Open an issue or reach out on GitHub.
